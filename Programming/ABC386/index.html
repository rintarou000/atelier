    • 問題の概要：
    文字列「ABCDE」からできる、空でない部分文字列31通りがある。
    入力として、各文字に対応する点数が与えられる。
    部分文字列の持つ点数の合計が高い順から出力せよ。
    ただし同点の場合、文字列が辞書順で小さいほうから出力せよ。

    分析：
    各文字を「選ぶ」か「選ばないか」かの２通りを取る。
    5文字の場合、25 = 32通り を経て部分文字列が完成する。
    まったく文字を選ばない空の場合を除いて、考えるべきは31通りある。

    方針：
    31通りなので全探索をする。
    「0」を「選ばない」、「1」を「選ぶ」ことに対応させる。
    2進数を0から32までつくる。
    2進数1桁目をA問題、2桁目をB問題のように対応させる。
    各部分文字列と点数のリストを作る。
    部分文字列を辞書の小さい順に並べる。
    得点を大きい順に並べる

    つまづいたところ：Pythonでの実装
    5桁の2進数を格納したい
    ⇒format(i,"b").zfill(5)
    2次元リストD=[文字列,整数]の各要素に着目してソートしたい
    ⇒D.sort(key=lambda x: (x[0]))
    ⇒D.sort(key=lambda x: (x[1]), reverse= True)
